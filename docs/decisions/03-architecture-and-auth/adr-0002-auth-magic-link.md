# ADR-0002: MVP 인증은 이메일 매직링크

| 항목 | 내용 |
|------|------|
| 상태 | ⚠️ 부분 대체 (→ ADR-0008) |
| 날짜 | 2026-06-18 |
| 단계 | Inception |

## 맥락 (Context)

MVP 로그인 수단을 정해야 했다. 후보: 이메일 매직링크, 구글/카카오/애플 로그인. 카카오는 이메일 동의 시 비즈 앱 전환, 애플은 iOS 배포 시 연 $99, 매직링크는 외부 OAuth 설정·심사가 없음.

## 결정 (Decision)

MVP 인증은 **이메일 매직링크** 하나로 시작하고, 소셜 로그인은 공개 배포 시 추가한다.

## 근거

- 인증은 핵심 가치가 아니라 관문. 최소 설정으로 시작.
- import 파이프라인이라는 진짜 핵심에 집중.

## 결과 (Consequences)

- (+) 최소 설정으로 즉시 시작.
- (−) 매직링크 이메일 수신 대기 UX.

## 갱신 (2026-07-08)

로컬 우선 전환(ADR-0007)으로 로컬 사용에는 인증이 불필요해졌다. **[ADR-0008](adr-0008-auth-only-for-server.md)** 이 이 결정을 **부분 대체** — 인증은 로컬 MVP의 P0가 아니라 **서버(백업/공유) 단계**로 이동한다. 수단(매직링크 등)은 Phase 2에서 확정.
