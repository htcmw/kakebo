## 설계 원칙

- 공동/멤버 색 구분: 공동=teal, 규관=blue, 윤선=pink. 이체=중립 회색.
- 이체·할부는 배지로 명시(⇄ 이체·지출 제외, N개월 할부/회차).
- 한국어·KRW·국내 날짜 형식(NFR-I18N-01).

## 이체·이중집계 방지 UI (ADR-0004/0005)

- 대시보드: 이체·카드대금 합계를 "지출 집계 제외"로 안내.
- 거래: 이체 행은 별도 표기(출금→입금/카드), 지출 제외 배지.
- 이체 관리 모달: ① 이체 짝 맞추기(출금↔입금/카드 연결) ② 이체 자동 규칙(키워드) ③ 일괄 이체 표시.
- 가져오기: 카드대금 의심 건 자동 '이체·제외' 플래그, 확정 전 조정.

## 할부 UI (ADR-0006)

- 거래 화면 "할부 지출 기준" 토글: 구매 시점 ↔ 월 청구.
- 편집 모달: 할부 개월 선택 + 월별 스케줄 미리보기.

## 미결/후속

- **모바일 반응형 미설계** (NFR-PLT-01). 데스크톱 목업 → 모바일 레이아웃 후속 필요.
- 접근성(NFR-A11Y, 대비·키보드)은 MVP 이후 강화.
